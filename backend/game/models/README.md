# Game Models Directory

This directory contains the Django models for all game entities.

- `entity.py`: Base entity class for all game objects.
- `player.py`: Player character model with stats and inventory.
- `room.py`: Room and Exit models for world representation.
- `item.py`: Item model with equipment and consumables.
- `npc.py`: NPC model with AI, combat, and merchant capabilities.
- `zone.py`: Zone/area grouping model.
- `quest.py`: Quest and PlayerQuest models.
