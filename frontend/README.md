# Frontend Directory

This directory contains all the source code for the MUD client. This includes the user interface and any other client-side logic.

## Files

- `package.json`: This file lists the project dependencies and scripts.
- `src/components/GameTerminal.js`: This is the main component for the game's terminal interface.
- `src/components/PlayerStats.js`: This component displays the player's stats.
- `README.md`: This file.
